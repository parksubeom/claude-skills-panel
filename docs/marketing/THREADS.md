# 짧은 thread (3개) — V1

X 7-tweet thread보다 짧은 버전. Threads는 thread 자체 효과가 X보다 약하니 3개로 충분.

## 🇺🇸 영문

### 1/3
```
Built a VS Code/Cursor panel for Claude Code that auto-finds every
slash command on your machine. Click to fire, or keys 1-6 for the
top six. Plus a /plugin marketplace browser inside (Claude Code
itself doesn't have one).

Made with Claude Code's full-flow skill, ~1 week.
```
첨부: 라인업

### 2/3
```
The pattern that surprised me while building it:

Most users concentrate ~80% of their slash command use in 6
commands. The other 40 are situational — findable, not memorable.

Once I labeled the categories, the panel basically wrote itself:
auto-discovery + grid + Quick Bar.
```

### 3/3
```
The fun part: the buddy character now branches into 1 of 10 RPG
classes based on your most-used category. /git-* heavy → Gitto the
Ninja. /code-review → Testra the Paladin.

All gamification toggles off if you just want a clean launcher.

Free, MIT: github.com/parksubeom/claude-skills-panel
```
첨부: 라인업

# 게시 팁 (Threads 특화)

| 행위 | 효과 |
|---|---|
| 본문 링크 | ✅ 페널티 적음 (X와 다름) |
| 첫 줄 hook | ✅ 중요 (스크롤 멈추기) |
| 이미지 1장 | ✅ 임프레션 ~2배 (X만큼 큰 차이는 아님) |
| 해시태그 | ❌ 효과 거의 없음 |
| 답글 빠르게 | ✅ 알고리즘 가속 (X와 동일) |
| 다른 Threads 글에 답글 | ✅ Threads는 답글 활동도 본인 노출에 영향 |
| Instagram 연동 | ✅ Instagram 팔로워에게도 도달 |
