## What

A free **VS Code / Cursor extension** that gives Claude Code users a graphical home for their slash commands.

## Who

Claude Code users with **5+ slash commands** (which is most users after installing one plugin).

## Why

Claude Code itself has no UI for browsing the slash commands you've installed. Once you have superpowers + code-review + your own custom commands, you've got 50+ entries you can't see.

This panel scans `~/.claude/`, project-level `.claude/`, and every plugin you've installed via `/plugin install`, and shows them all as a clickable, fuzzy-searchable card grid.

## How

- **Click a card** → command goes to clipboard, or auto-paste, or active terminal
- **Quick Bar with keys 1–6** → fire your top six from anywhere
- **🛒 built-in Plugin Browser** → 243+ official plugins searchable, one-click install
- **Fuzzy search + keyboard nav** (`↓ Enter`, or `/foo` for raw slash)
- **3 pixel themes** (Dark, Retro CRT, Gameboy LCD) · **3 languages** (en/ko/ja)
- **Optional pixel-art game**: 10 RPG-class buddies that branch from your usage pattern. All gamification toggles off in one click.
