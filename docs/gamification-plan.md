# Gamification Plan

도트 게이미피케이션 컨셉으로의 전환 계획. 4 페이즈, 단계별 커밋.

## 원칙
1. **본업 우선** — 스킬 빠르게 트리거하는 본 기능을 방해하지 않는다.
2. **항상 끄기 가능** — 사운드/애니메이션은 토글 제공.
3. **점진적** — A→B→C→D 순서로, 각 페이즈 끝나면 커밋.
4. **데이터 가벼움** — 진행도/카운트는 기존 `~/.claude/skills-panel-config.json` 에 누적 저장.

---

## Phase A — 시각 (8-bit UI)
**목표**: 첫 화면에서 "도트 게임" 분위기 즉시 인지.

- [ ] 픽셀 폰트 임베드: DungGeunMo (한글) + Press Start 2P (영문 fallback)
- [ ] 배경: 격자(grid) + 살짝 어두운 vignette
- [ ] 스킬 카드: chamfer 모서리(꺾인 사각형) + 두꺼운 border 강화
- [ ] CRT 효과: scanlines 오버레이 (옵션)
- [ ] 글로우: 호버 시 픽셀 sparkle 파티클
- [ ] 클릭 효과: 화면 짧은 shake
- [ ] 8-bit 사운드: hover beep / click bleep / copy chime (Web Audio, 토글)
- [ ] 토스트 → 게임 다이얼로그 박스 풍 (border 코너 장식)
- [ ] 등장 애니메이션: 카드 순차 fade-up

## Phase B — 진행도 / 마스터리
**목표**: 사용할수록 카드가 "성장"하는 시각적 보상.

- [ ] `~/.claude/skills-panel-config.json` 에 `usage` 필드: `{ count, lastUsed, firstUsed }`
- [ ] 클릭(=copy) 시마다 +1
- [ ] 마스터리 등급: 0(미사용) / 1(1~4) / 2(5~14) / 3(15~49) / 4(50~149) / 5(150+)
- [ ] 카드 하단에 별 막대 ★★★☆☆ 표기 (5단계)
- [ ] 등급 상승 시 토스트: "🎉 today LV.3 달성"
- [ ] 데일리 스트릭: 오늘 최소 1개 이상 사용 → 푸터에 🔥 N일
- [ ] 히트맵(옵션): 최근 7일 사용 패턴 시각화

## Phase C — Quick Bar (장비 슬롯)
**목표**: 자주 쓰는 스킬 즉시 트리거 — 게임 단축키 슬롯 메타포.

- [ ] 상단 가로 바: 6개 슬롯 (1~6 키)
- [ ] 드래그 앤 드롭으로 카드를 슬롯에 "장착"
- [ ] 슬롯의 키 입력 → 즉시 `/skill` 복사
- [ ] 빈 슬롯 시각: 회색 빗금 박스
- [ ] 슬롯 우클릭 → 비우기

## Phase D — 메타게임 (업적 / 위클리)
**목표**: 장기 동기 유발.

- [ ] 업적 정의 데이터: `data/achievements.json`
  - 첫 사용 / 10번 / 5종 다른 스킬 / 7일 연속 / 한 스킬 LV.5 등
- [ ] 달성 시 토스트 + `config.json` 의 `achievements` 배열에 기록
- [ ] 업적 보드 페이지(설정 모달 또는 별도 패널)
- [ ] 위클리 리포트: 가장 많이 쓴 스킬 / 신기록 / 미사용 추천 1개
- [ ] (옵션) Easter egg — 특정 입력 시 픽셀 캐릭터 등장
